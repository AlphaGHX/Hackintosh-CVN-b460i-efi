## 使用前

* 基于opencore 0.6.5
* 支持macOS BigSur 11.2
* 支持七彩虹CVN-b460i
* 支持Intel10代CPU，UHD 630 GPU, 独显未做支持！
* 请自行更改本机序列号等信息

## 目前正常功能

* WIFI正常（Intel AX200）
* 蓝牙正常
* 有线正常
* HDMI & DP正常
* 睡眠能用，但是不太正常。
